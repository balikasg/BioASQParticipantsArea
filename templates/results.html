Test Results

Currently, there are no results available. As soon as a new test set will be released the "Results" section will be initialised in the following way:

  • The evaluation function will be triggered after the expiration of the test set. The System Descriptions of the systems that uploaded results will fill in the Results sets of tables.
  • Each test set will form set of tables, a table with the Flat Measures and another table with the Hierarchical Measures.
  • The Results tables will be updated in a weekly basis.
  • For the assessment of the participating systems one flat, Micro F-Measure and one hierarchical, Lowest Common Ancestor-Symmetric Difference Loss measure will be used. For completeness, we will compute more measures from each family, presented below.

The flat measures that are going to be calculated are:

  1. Accuracy (Acc.)
  2. Example Based Precision (EBP)
  3. Example Based Recall (EBR)
  4. Example Based F-Measure (EBF)
  5. Macro Precision (MaP)
  6. Macro Recall (MaR)
  7. Macro F-Measure (MaF)
  8. Micro Precision (MiP)
  9. Micro Recall (MiR)
  10. Micro F-Measure (MiF)

The Hierarchical Measures that are going to be calculated are:

  1. Hierarchical Precision (H. Prec)
  2. Hierarchical Recall (H. Recall)
  3. Hierarchical F-Measure(H. F-Measure)
  4. Lowest Common Ancestor-Symmetric Difference Loss (LCA-SDL)
