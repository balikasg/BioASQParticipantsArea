Frequently Asked Questions

Task 2A

  1. How many articles does the training set include?
    2. There are about 12 milion articles in the training set. In every article there is an abstract and in some of them the full text is available.

  2. Which are the classes that are going to be used during classification?
    3. MeSH (Medical Subject Headings) is the vocabulary that is going to be used for classification.

  3. How many labels does each article usually receive?
    4. Usually articles receive around 10-15 labels.

  4. In what formats are the training and the test data going to be provided during the challenge?
    5. Training and test data are going to be provided both in raw text and in vectorized description. The vectorized description will be distributed as a Lucene Index. Detailed description of the fields of the index and the vectorization tools is provided analytically.

  5. From which journals are the test articles going to be selected?
    6. The list of the journals can be found here.
